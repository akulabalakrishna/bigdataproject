# Demo Runbook Manual Screenshot Capture

Since autonomous browser automation is not available in the current terminal sandbox, please manually open the following URLs on your host machine to capture the final UI screenshots required for your Big Data Lab Deliverable 3 submission evidence:

1. **Spark UI home page:** [http://localhost:8080](http://localhost:8080)
   *Save as: screenshots/spark_ui.png*
2. **MLflow UI:** [http://localhost:5000](http://localhost:5000)
   *Save as: screenshots/mlflow_ui.png*
3. **Airflow UI or DAG list page:** [http://localhost:8085](http://localhost:8085)
   *Save as: screenshots/airflow_ui.png*
4. **Prometheus query page or API status:** [http://localhost:9090](http://localhost:9090)
   *Save as: screenshots/prometheus_ui.png*
5. **Grafana home/dashboard page:** [http://localhost:3001](http://localhost:3001)
   *Save as: screenshots/grafana_ui.png*
6. **BentoML docs/ready page:** [http://localhost:3002/docs](http://localhost:3002/docs)
   *Save as: screenshots/bentoml_api.png*

Remember:
- Do not screenshot raw patient rows.
- Do not screenshot raw MIMIC files.
- Keep the screenshots strictly focused on the system UI, dashboards, pipelines, and health statuses.
